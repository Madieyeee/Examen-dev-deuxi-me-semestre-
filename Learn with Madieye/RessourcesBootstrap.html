Introduction à Bootstrap

Bootstrap est un framework CSS populaire qui facilite la création de sites web réactifs et élégants. Il inclut des styles CSS prédéfinis et des composants JavaScript.

Commencer avec Bootstrap :

Installation

Vous pouvez installer Bootstrap de plusieurs manières. La méthode la plus simple consiste à utiliser un CDN (Content Delivery Network).

<!-- CSS de Bootstrap -->
<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript de Bootstrap -->
<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.2/dist/umd/popper.min.js"></script>
<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

Mise en Page avec Bootstrap :

Le Système de Grille

Le système de grille de Bootstrap est basé sur une grille de 12 colonnes. Voici un exemple de disposition de base :

<div class="container">
    <div class="row">
        <div class="col">Colonne 1</div>
        <div class="col">Colonne 2</div>
        <div class="col">Colonne 3</div>
    </div>
</div>

Les Conteneurs

Les conteneurs sont utilisés pour centrer et donner un maximum de largeur aux contenus de la page. Vous pouvez utiliser .container ou .container-fluid.

<div class="container">
    Contenu centré et avec une largeur maximale fixe.
</div>
<div class="container-fluid">
    Contenu centré et avec une largeur maximale de 100%.
</div>

Composants Bootstrap :

Les Boutons

Bootstrap inclut des styles prédéfinis pour les boutons. Vous pouvez appliquer différentes classes pour changer leur apparence.

<button type="button" class="btn btn-primary">Bouton primaire</button>
<button type="button" class="btn btn-secondary">Bouton secondaire</button>
<button type="button" class="btn btn-success">Bouton succès</button>
<button type="button" class="btn btn-danger">Bouton danger</button>

Les Alertes

Les alertes sont des messages contextuels qui peuvent être utilisés pour afficher des informations importantes.

<div class="alert alert-primary" role="alert">
    Alerte primaire !
</div>
<div class="alert alert-secondary" role="alert">
    Alerte secondaire !
</div>

Les Cartes

Les cartes sont des conteneurs flexibles et extensibles pour afficher du contenu varié.

<div class="card" style="width: 18rem;">
    <img src="..." class="card-img-top" alt="...">
    <div class="card-body">
        <h5 class="card-title">Titre de la carte</h5>
        <p class="card-text">Un texte rapide pour constituer le corps de la carte.</p>
        <a href="#" class="btn btn-primary">Aller quelque part</a>
    </div>
</div>

Utilitaires Bootstrap :

Classes d'Espacement

Bootstrap offre des classes utilitaires pour gérer les marges et les espacements. Par exemple :

<div class="mb-3">Marge inférieure de 3 unités</div>
<div class="mt-5">Marge supérieure de 5 unités</div>

Classes de Couleur

Vous pouvez appliquer des couleurs prédéfinies à différents éléments.

<p class="text-primary">Texte primaire</p>
<p class="text-danger">Texte danger</p>
<p class="bg-warning">Fond d'avertissement</p>

Utilitaires Responsives :

Classes Visibilité

Ces classes permettent de cacher ou montrer des éléments selon la taille de l'écran.

<div class="d-none d-md-block">
    Visible seulement sur les écrans moyens et plus grands.
</div>
<div class="d-sm-none">
    Caché sur les écrans petits et plus petits.
</div>
