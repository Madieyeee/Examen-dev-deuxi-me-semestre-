Introduction au HTML


HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il permet de structurer et de formater le contenu des sites web.

Éléments de base :

Les Balises HTML

Les balises HTML sont les éléments de base du langage HTML. Elles sont utilisées pour créer les différents composants d'une page web.

  • <html> : La balise racine de tout document HTML.
  • <head> : Contient des métadonnées sur le document HTML.
  • <body> : Contient le contenu principal de la page web.

Les Titres

Les balises de titre sont utilisées pour créer des titres de différentes tailles. Il y a six niveaux de titres, de <h1> à <h6>, <h1> étant le plus important.

<h1>Titre de niveau 1</h1>
<h2>Titre de niveau 2</h2>
<h3>Titre de niveau 3</h3>
<h4>Titre de niveau 4</h4>
<h5>Titre de niveau 5</h5>
<h6>Titre de niveau 6</h6>

Les Paragraphes

La balise <p> est utilisée pour créer des paragraphes de texte.

<p>Ceci est un paragraphe de texte.</p>

Les Liens

Les liens hypertextes sont créés avec la balise <a>. Ils permettent de naviguer d'une page à l'autre.

<a href="https://www.example.com">Visiter Example.com</a>

Les Images

Les images sont insérées avec la balise <img>. Elle nécessite un attribut src pour spécifier la source de l'image.

<img src="image.jpg" alt="Description de l'image">

Formulaires et Tables :

Formulaires

Les formulaires permettent aux utilisateurs de saisir des données. Ils sont créés avec la balise <form>.

<form action="/submit" method="post">
    <label for="name">Nom :</label>
    <input type="text" id="name" name="name"><br><br>
    <label for="email">Email :</label>
    <input type="email" id="email" name="email"><br><br>
    <textarea id="message" name="message" rows="4" cols="50">Votre message ici...</textarea><br><br>
    <button type="submit">Envoyer</button>
</form>

Tables

Les tables sont utilisées pour afficher des données tabulaires. Elles sont créées avec les balises <table>, <tr>, <th> et <td>.

<table>
    <tr>
        <th>Nom</th>
        <th>Âge</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>24</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
    </tr>
</table>

Multimédia :

Vidéo

Les vidéos peuvent être intégrées avec la balise <video>. Elle nécessite les attributs src et controls.

<video src="video.mp4" controls>Votre navigateur ne supporte pas la balise vidéo.</video>

Audio

Les fichiers audio peuvent être intégrés avec la balise <audio>. Elle nécessite l'attribut src et controls.

<audio src="audio.mp3" controls>Votre navigateur ne supporte pas la balise audio.</audio>

Éléments Sémantiques :

Les Balises Sémantiques

Les balises sémantiques sont utilisées pour donner du sens au contenu HTML. Elles améliorent l'accessibilité et le SEO.

  • <header> : Représente l'en-tête d'un document ou d'une section.
  • <nav> : Représente une section de navigation.
  • <article> : Représente un contenu indépendant et autonome.
  • <section> : Représente une section générique d'un document.
  • <footer> : Représente le pied de page d'un document ou d'une section.

Éléments de Structure :

Divisions

<div>
    <h2>Titre dans une div</h2>
    <p>Paragraphe dans une div.</p>
</div>

Attributs Globaux :

id

<p id="paragraphe-unique">Paragraphe avec un ID.</p>

class

<p class="paragraphe-style">Paragraphe avec une classe.</p>

style

<p style="color:blue;">Paragraphe avec style en ligne.</p>
