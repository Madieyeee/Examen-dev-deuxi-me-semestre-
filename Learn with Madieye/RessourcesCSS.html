Introduction au CSS

CSS (Cascading Style Sheets) est un langage de feuille de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler la mise en forme et la disposition des éléments sur une page web.

Syntaxe et Sélecteurs

La Syntaxe CSS

Un fichier CSS est constitué de règles. Chaque règle est composée d'un sélecteur et d'un bloc de déclarations. Voici un exemple :

sélecteur {
    propriété: valeur;
}

Les Sélecteurs CSS

Les sélecteurs sont utilisés pour sélectionner les éléments HTML que vous souhaitez styliser. Voici quelques exemples de sélecteurs courants :

  • Sélecteur de type : Sélectionne tous les éléments d'un type donné. Ex : p
  • Sélecteur de classe : Sélectionne les éléments avec une classe spécifique. Ex : .classe
  • Sélecteur d'identifiant : Sélectionne l'élément avec un identifiant spécifique. Ex : #identifiant

Le Modèle de Boîte

Comprendre le Modèle de Boîte

Le modèle de boîte est un concept fondamental en CSS. Chaque élément est représenté comme une boîte rectangulaire. Voici les composants de la boîte :

  • Contenu : Le contenu réel de l'élément, comme le texte ou l'image.
  • Padding : L'espace autour du contenu, à l'intérieur de la bordure.
  • Bordure : La bordure entourant le padding (et le contenu).
  • Marges : L'espace autour de la bordure, à l'extérieur de l'élément.
/* Exemple de CSS pour le modèle de boîte */
div {
    width: 300px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

Disposition et Positionnement

Le Modèle de Boîte Flexible (Flexbox)

Flexbox est un modèle de mise en page qui permet de créer des mises en page complexes avec un minimum de code. Voici un exemple :

/* Container flex */
.container {
    display: flex;
    justify-content: space-between;
}

/* Élément flex */
.item {
    flex: 1;
    padding: 10px;
}

Grille CSS (CSS Grid)

La grille CSS est un autre système de mise en page puissant. Elle permet de créer des grilles bidimensionnelles. Voici un exemple :

/* Container de la grille */
.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
}

/* Élément de la grille */
.grid-item {
    padding: 20px;
    background-color: lightgrey;
    border: 1px solid black;
}

Typographie

Polices de Caractères

Vous pouvez spécifier les polices de caractères à utiliser sur votre page web. Voici un exemple :

body {
    font-family: Arial, sans-serif;
}
h1 {
    font-family: 'Times New Roman', Times, serif;
}

Taille de la Police

Vous pouvez contrôler la taille des polices avec la propriété font-size. Voici un exemple :

p {
    font-size: 16px;
}
h1 {
    font-size: 2em;
}

Couleur de la Police

La couleur de la police peut être modifiée avec la propriété color. Voici un exemple :

p {
    color: #333;
}
h1 {
    color: blue;
}

Design Responsive

Médias Queries

Les médias queries sont utilisées pour appliquer des styles spécifiques en fonction de la taille de l'écran. Voici un exemple :

/* Pour les écrans de 600px et moins */
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Animations et Transitions

Transitions CSS

Les transitions permettent de changer progressivement d'une valeur de style à une autre. Voici un exemple :

button {
    background-color: blue;
    transition: background-color 0.5s ease;
}

button:hover {
    background-color: green;
}

Animations CSS

Les animations permettent de créer des effets de mouvement plus complexes. Voici un exemple :

@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

div {
    width: 100px;
    height: 100px;
    animation-name: example;
    animation-duration: 4s;
}
