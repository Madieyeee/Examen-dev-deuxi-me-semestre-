Introduction au JavaScript

JavaScript est un langage de programmation utilisé pour créer des contenus dynamiques et interactifs sur les pages web. Il est essentiel pour le développement web moderne.

Les Bases de JavaScript

Syntaxe de Base

JavaScript utilise une syntaxe simple et intuitive. Voici un exemple de code JavaScript :

<script>
    console.log('Hello, world!');
</script>

Les Variables

Les variables en JavaScript sont utilisées pour stocker des données. Vous pouvez les déclarer avec var, let ou const.

let message = 'Hello, world!';
let age = 20;
const pi = 3.14;

Les Types de Données

JavaScript prend en charge différents types de données, notamment les chaînes de caractères, les nombres, les objets, et les booléens.

let name = 'Anne';
let nombre = 42;
let boolean = true;
let objet = { prenom: "Madieye", age: 20 };
let tableau = [1, 2, 3];

Les Fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent des tâches spécifiques.

function direBonjour() {
    console.log("Bonjour tout le monde!");
}
direBonjour();

Les Conditions

Les instructions conditionnelles permettent d'exécuter du code différent en fonction de certaines conditions.

let age = 18;
if (age >= 18) {
    console.log("Vous êtes majeur.");
} else {
    console.log("Vous êtes mineur.");
}

Les Boucles

Les oucles permettent d'exécuter du code plusieurs fois.

for (let i = 0; i < 5; i++) {
    console.log("i vaut " + i);
}

let j = 0;
while (j < 5) {
    console.log("j vaut " + j);
    j++;
}

Les Événements

Les événements permettent de rendre une page interactive en répondant aux actions de l'utilisateur.

<button id="monBouton">Cliquez-moi</button>

<script>
document.getElementById("monBouton").addEventListener("click", function() {
    alert("Bouton cliqué!");
});
</script>

Manipulation du DOM

Sélectionner des Éléments

JavaScript permet de sélectionner et de manipuler les éléments HTML en utilisant le DOM (Document Object Model).

let element = document.getElementById('myElement');
let elements = document.getElementsByClassName('myClass');
let elementsByTag = document.getElementsByTagName('p');
let queryElement = document.querySelector('.myClass');

Modifier des Éléments

Vous pouvez modifier le contenu et les styles des éléments HTML avec JavaScript.

let element = document.getElementById('myElement');
element.textContent = 'Nouveau contenu';
element.style.color = 'blue';

Les Tableaux et les Objet

Les Objets

Les objets permettent de regrouper des valeurs et des fonctions qui leur sont associées.

let personne = {
    prenom: "Madieye",
    age: 20,
    saluer: function() {
        console.log("Bonjour, je m'appelle " + this.prenom);
    }
};
personne.saluer();

Les Tableaux

Les tableaux permettent de stocker plusieurs valeurs dans une seule variable.

let fruits = ["Pomme", "Banane", "Cerise"];
    console.log(fruits[0]); // Affiche "Pomme"
                    
    fruits.push("Orange");
    console.log(fruits.length); // Affiche 4
